# ML Uppgift 1 – California Housing

Detta repository innehåller lösningen till ML Uppgift 1 inom kursen Machine Learning.

Uppgiften bygger på California Housing-datasetet och syftet är att undersöka hur maskininlärning kan användas som beslutsstöd för att analysera bostadsområden.

## Innehåll
- Jupyter Notebook med kod, analyser och visualiseringar
- PDF-rapport (1–2 sidor)
- Dataset (housing.csv)

## Spår
- Spår A – Regression (förutsägelse av median_house_value)

## Körinstruktioner
1. Öppna notebooken `ML_Uppgift1_Ime_Prezime.ipynb`
2. Välj "Restart & Run All" för att köra hela analysen

## Miljö
- Python version: 3.13
